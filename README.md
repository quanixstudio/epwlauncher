# Evolved Perfect World Launcher

Evolved Perfect World Launcher is a tool that makes it easier for you to log into the game.
With only 1 click it will launch the game and log into any toon you want.

When you open the tool for the first time it should show you the Settings.
Here you first need to select the location of the "element" folder of the game.
It will tell you if you selected the wrong folder.
You can ignore the other settings for now.

After you selected the game folder you can start adding accounts.
To do this, click the Accounts button at the bottom and fill in the fields at Add Account.
All the accounts that you add will show on the left side of the tool with the Display Name.
The username and password are the ones you use to login to the game.

When you have added 1 or multiple accounts it is time to add your toons to them.
This can be done by clicking the Toons button at the bottom.
Choose the account, choose the class, enter the name of your toon and click on Add Toon.
The Class option here is only used for the icons in the list with toons.

After you done all of this you can go back to settings and set a default account to show on startup if you prefer.

You should now be able to click on an account at the left and click the arrow at a toon to launch the game and log into that toon.
